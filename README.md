# NAME

Aobabot - 슬랙 봇

# SYNOPSIS

![](https://pbs.twimg.com/media/Cs7J0NTVMAAEavT.jpg:large)

```sh
# 저장소 클론
$ git clone https://github.com/unstabler/aobabot.git
# 의존 모듈 설치 (cpanm 필요)
$ cpanm installdeps
# 설정 파일 복사하고 편집하기
$ cp config.json.sample config.json
# 아오바 봇 실행
$ perl aobabot.pl
```
